# Sistem Pencatatan Toko - Toko ABB

Aplikasi web sederhana berbasis PHP dan MySQL
untuk pencatatan data barang pada Toko ABB.

## Fitur
- Login Admin
- CRUD Data Barang
- Sistem berbasis web

## Teknologi
- PHP Native
- MySQL
- HTML & CSS

## Cara Menjalankan
1. Letakkan folder di htdocs
2. Jalankan Apache & MySQL
3. Import database
4. Akses melalui browser

## Catatan
Database tidak disertakan dalam repository GitHub.
